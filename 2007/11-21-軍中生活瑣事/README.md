今天收假

我試著用走的從台南火車站走到營區

背著包包大步前進的話大概要半個小時才能到

還算蠻遠的...@_@

不過一直麻煩別人載也不太好

我第一次遇到這種狀況

不過一直到集合點名

我都沒有看到來督導的長官

另外

資訊業務又多一個候補的學弟了
