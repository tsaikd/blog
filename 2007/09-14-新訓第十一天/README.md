早上在中山室吃早餐，上午的課程是在餐廳聽自願役的宣導，我都搬過去了

還要一起去聽廢話....

    今天是莒光日，下午就一直在寫作文，還有理髮，明明就沒啥頭髮了，還要

理，真是無言，大概是明天要懇親會，所以才這樣麻煩吧....

    餐廳的桌椅在下午的時候就陸續搬到外面布置懇親的場地，所以我們的晚餐

就只好在中山室吃了。

    我們自願役的吃飽飯後就回營部連寢室聊天打屁，美其名為準備智力測驗的

考試，實際上真的有在看書的人應該不太多，不知道其他義務役的人在做什麼？
